# Благотворительный фонд Шаңырақ

Сайт благотворительного фонда с категориями помощи и информацией о сборах.

## Структура проекта

- `index.html` - Основной HTML файл
- `app.js` - Главный React компонент
- `components/` - React компоненты
  - `Header.js` - Шапка с логотипом
  - `SearchBar.js` - Поиск
  - `CategoryCards.js` - Карточки категорий
  - `TotalAmount.js` - Сумма сбора
  - `PartnerFunds.js` - Фонды партнеры
  - `AppDownload.js` - Скачать приложение

## Дизайн система

- **Основной цвет**: #2d7d6e (зеленый)
- **Вторичный цвет**: #f5e6e8 (светло-розовый)
- **Шрифт**: System fonts
- **Стиль**: Современный, с закругленными углами

## Функции

- Адаптивный дизайн
- Карточки категорий помощи
- Отображение суммы сборов
- Список фондов партнеров
- Кнопки скачивания приложения

## Технологии

- React 18
- Tailwind CSS
- Lucide Icons
