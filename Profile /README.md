# Шанырак - Благотворительная платформа

Веб-приложение благотворительного фонда "Шанырак" на Trickle хостинге.

## 🎯 Особенности

- ✅ Работает на Trickle хостинге
- ✅ Встроенная база данных Trickle
- ✅ React 18 + CDN (без сборки)
- ✅ Экспорт данных в CSV/JSON

## 🚀 Использование

Откройте любую страницу на Trickle хостинге:
- `index.html` - Главная страница
- `export.html` - Экспорт базы данных
- `reports.html` - Отчеты
- `admin.html` - Админ-панель

## 📊 База данных

- `charity_beneficiary` - подопечные
- `admin_user` - администраторы
- `partner_fund` - фонды-партнеры
- `company_profile` - спонсоры
